# **App Name**: EduFinance

## Core Features:

- Secure Sign-Up & Login: Secure authentication using phone number and OTP to minimize profile information stored.
- Manual Expense Tracking: Allow users to manually add expenses with categories, amounts, optional notes, date, and time.
- Monthly Budget & Goal Planning: Enable users to set monthly budgets and short-term financial goals, tracking spending, remaining amounts, and progress.
- Visual Analytics: Provide pie charts for category-wise spending and bar/line charts for daily/weekly spending within the current month.
- Alerts and Nudges: Send local notifications when spending crosses 80% and 100% of the monthly budget and offer rule-based saving tips.
- Scholarship Directory: List scholarships and schemes, with filtering by state, income, and category. Display static content.
- AI Powered Saving Suggestion: Based on spending habits, our AI tool suggests savings based on a library of potential savings.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and reliability.
- Background color: Light blue (#E3F2FD), creating a clean and calming atmosphere.
- Accent color: Teal (#009688) to highlight important elements and CTAs.
- Body and headline font: 'PT Sans' for readability and a modern, friendly feel.
- Use flat, consistent icons to represent spending categories and features.
- Simple, card-based layout for clear data presentation and ease of navigation.
- Subtle animations and transitions to enhance UX and provide feedback during interactions.
